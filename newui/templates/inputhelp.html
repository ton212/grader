Python

Write a function run(). Use the yield keyword to return a test case. The generator must not be endless.

Example: 

def run():
    for i in ["madoka", "Homura", "Mami", "Kyubey", "Sayaka", "kyoko"]:
        yield i

PHP

Write a function run() which return every test cases as an array.

Example: 

<?php
function run(){
    return array("madoka", "Homura", "Mami", "Kyubey", "Sayaka", "kyoko");
}

Java

Write a method with signature public static void run(List<String> out) inside a public class without package.

Add every test cases as a String to the out variable provided.

Avoid using Java as the compilation process is slower than other languages.

Example: 

import java.util.List;

public class HelloWorldInput{
    public static void run(List<String> out){
        out.add("Madoka");
        out.add("Homura");
        out.add("mami");
        out.add("kyouko");
        out.add("kyUbeY");
    }
}
